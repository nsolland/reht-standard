# Public Repository Boundaries

This repository is the public, static specification surface for REHT.

## Included

- public terminology
- versioned contracts
- reference JSON Schemas
- non-production examples
- architecture descriptions
- conformance language
- public change history

## Excluded

- VALO production source code
- VAIG evaluation logic
- REHT proprietary decision implementation
- BARO collection or analysis code
- execution kernel code
- internal thresholds, weights and risk models
- customer integrations and customer data
- credentials, infrastructure and deployment configuration
- operational runbooks
- unpublished research and patent material

## Separation rule

No file may be copied from a private production repository without an explicit public-release review covering intellectual property, security, customer confidentiality and dependency licensing.

Schema compatibility does not imply implementation equivalence, certification, endorsement or compliance.
